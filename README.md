# 🍔 Late Delivery Analysis - Food Ordering App

## 📌 Business Problem
Customer complaints increased in the last 2 weeks due to late food deliveries.

## 🎯 Objective
Identify the root causes of late deliveries and suggest data-driven business solutions.

## 🧠 Data Analytics Approach
This project follows the **6 stages of Data Analytics**:

1. Ask
2. Prepare
3. Process
4. Analyze
5. Share
6. Act

## 📊 Key KPIs
- Average Delivery Time
- On-Time Delivery %
- Complaint Rate %
- Peak Hour Delay %
- Region-wise Delay %
- Restaurant Prep Time

## 🔍 Key Insights
- Peak dinner hours show maximum delays
- Some regions have rider shortages
- Restaurant prep time contributes heavily
- ETA prediction needs improvement

## ✅ Suggested Actions
- Add delivery partners in peak hours
- Improve ETA prediction
- Limit orders during overload
- Optimize restaurant prep workflow
